<h1>ClockApp — 透明なオーバーレイ時計</h1>

<p>Windows用のシンプルなオーバーレイ時計。常時最前面表示、右下ドラッグでサイズ変更、ウインドウサイズに合わせてフォント自動スケール、タスクトレイ常駐、カウントダウン・指定時刻までの逆算に対応。</p>

<hr />

<h2>主な機能</h2>
<ul>
  <li>現在時刻表示 / カウントダウン / 指定時刻までの逆算</li>
  <li>右下隅ドラッグでサイズ変更（枠なしでもリサイズ可能）</li>
  <li>ウインドウサイズに応じてタイトルと時計のフォントを自動スケール</li>
  <li>常に手前に表示（オン/オフ切替可）</li>
  <li>背景透過（TransparencyKey=WhiteSmoke 切替）</li>
  <li>文字色変更（タイトル・時計を同色で変更）</li>
  <li>透明度（Opacity）をスライダー操作で即時反映、キャンセルで元に戻す</li>
  <li>言語切替（日本語/英語）</li>
  <li>「初期設定に戻す」で起動直後の状態へ一括リセット（フォント種類・サイズ・色・不透明度・各設定を含む）</li>
  <li>タスクトレイ常駐・最小化時はタスクトレイへ格納</li>
</ul>

<hr />

<h2>動作環境</h2>
<ul>
  <li>OS: Windows 10 / 11</li>
  <li>フレームワーク: WinForms（.NET Framework 4.7.2+ もしくは .NET 6/8 Windows）</li>
  <li>IDE: Visual Studio 2022 推奨</li>
</ul>

<hr />

<h2>ビルドと実行</h2>
<ol>
  <li>Visual Studio でソリューションを開く</li>
  <li>対象フレームワークを上記いずれかに設定</li>
  <li><code>ClockApp</code> をスタートアッププロジェクトに設定</li>
  <li>実行（Ctrl+F5）</li>
</ol>

<p><strong>アイコンの統一（任意）</strong><br />
フォームとタスクトレイのアイコンは <code>Form1.Designer.cs</code> の <code>this.Icon</code> に設定された ICO が使用されます。<br />
例：<code>D:\DataGitHub\Clock\Clock\Clock_icon.ico</code> をプロジェクトに追加し、フォームの Icon に割り当ててください。</p>

<hr />

<h2>使い方</h2>

<h3>表示とサイズ</h3>
<ul>
  <li>初回起動時は中央に黒背景で表示（Opacity=0.75）</li>
  <li>ウインドウ右下隅を左ドラッグでサイズ変更（フォントは自動的に拡大/縮小）</li>
  <li>タイトルと時刻ラベルをドラッグすると表示位置を移動</li>
</ul>

<h3>コンテキストメニュー（右クリック）</h3>
<p>項目は次の順で表示されます。</p>
<ol>
  <li>日本語表記（チェックで日本語。外すと英語）</li>
  <li>常に手前に表示（オン/オフ）</li>
  <li>ドラッグ移動を有効（オン/オフ）</li>
  <li>背景を透過（白煙色）<br />
    <small>チェック時は背景色と TransparencyKey を <code>WhiteSmoke</code> に設定します。文字色を背景と同色にすると見えなくなるため注意。</small>
  </li>
  <li>文字色を変更…（ColorDialog）</li>
  <li>透明度を数値で指定…（スライダー。動かした時点で即時反映／キャンセルで元に戻る）</li>
  <li>タイトル文を変更…（自由入力）</li>
  <li>タイトルの大きさフォントを変更…（FontDialog）</li>
  <li>時計の大きさフォントを変更…（FontDialog）</li>
  <li>現在の時刻を表示（時計モードへ切替）</li>
  <li>タイマーを設定（残り時間指定）…（分で設定）</li>
  <li>タイマーを設定（時間を指定して逆算）…（時刻で設定）</li>
  <li>初期設定に戻す（起動直後の状態へ一括リセット）</li>
  <li>アプリの終了</li>
</ol>

<h3>自動タイトルの仕様</h3>
<ul>
  <li>デフォルトのタイトルは <code>MM/dd/午前|午後</code>（英語時は <code>MM/dd/AM|PM</code>）</li>
  <li>時刻の進行で 12:00 に達すると 午後/PM、0:00 に達すると 午前/AM に自動切替</li>
  <li>日付変更時は <code>MM/dd</code> が自動更新</li>
  <li>ユーザーが「タイトル文を変更」で任意文字列にした場合は自動更新を停止（再び自動に戻すには、タイトルをデフォルト表記へ戻すか「初期設定に戻す」を実行）</li>
</ul>

<h3>タスクトレイ</h3>
<ul>
  <li>最小化時はタスクトレイへ格納</li>
  <li>トレイアイコンのダブルクリックで再表示</li>
  <li>トレイのコンテキストメニューはフォームと同一</li>
</ul>

<hr />

<h2>注意点</h2>
<ul>
  <li>背景透過を有効にしている場合、TransparencyKey と同じ色の文字は不可視になります。必要に応じて文字色を変更してください。</li>
  <li>「ドラッグ移動を有効」をオフにすると、ウインドウはクリックを透過します。再度オンにすると通常の操作に戻ります。</li>
  <li>透明度ダイアログはスライダー操作で即時反映しますが、キャンセルを押すと起動時の値へロールバックします。</li>
</ul>

<hr />

<h2>トラブルシュート</h2>
<ul>
  <li><strong>最前面にならない</strong>：コンテキストメニューの「常に手前に表示」にチェックがあるか確認。あるのに効かない場合は一度チェックを外して付け直してください。</li>
  <li><strong>文字が見えない</strong>：背景透過がオンで、文字色が <code>WhiteSmoke</code> になっていないか確認。別の色に変更してください。</li>
  <li><strong>サイズを変えても文字が小さい/大きい</strong>：右下隅からリサイズしてください。フォントは自動追従します。必要ならメニューからフォントを直接調整してください。</li>
  <li><strong>デザインを初期状態に戻したい</strong>：「初期設定に戻す」を実行。</li>
</ul>

<hr />

<h2>キーバインド / 操作</h2>
<ul>
  <li>タイトル/時計ラベルのドラッグ：ウインドウ移動</li>
  <li>ウインドウ右下ドラッグ：リサイズ（フォント自動スケール）</li>
  <li>右クリック：コンテキストメニュー</li>
</ul>
